# inbox/

Capture zone. Everything lands here first — quick notes, links, half-thoughts, screenshots, anything you don't want to lose.

Name captures `YYYY-MM-DD-slug.md` (e.g. `2026-07-11-idea-for-dashboard.md`) so old, unprocessed items are easy to spot.

Don't organize while capturing. Process later: move each item to `projects/`, `areas/`, `ideas/`, `data/`, or delete it.

Rule of thumb: if `inbox/` is empty, you're caught up.
