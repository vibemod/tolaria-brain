# 03-areas/

Ongoing responsibilities with no end date — standards you maintain, not things you finish.

Examples: health, personal finances, your role as team lead, the house, key relationships, a recurring habit.

Areas never get "done" — that's the whole test. If "done by when?" has an answer, it's a project. Areas often *spawn* projects: the area `health.md` might spawn the project `run-a-10k-in-october.md`. Keep the long-lived notes here; move the finishable effort to `02-projects/`.

One file (or folder) per area, holding the standard you uphold and notes you revisit.
