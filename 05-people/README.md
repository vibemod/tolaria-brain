# 05-people/

Notes on the people in my orbit — colleagues, collaborators, contacts. One file per person.

Name files after the person: `firstname-lastname.md` (e.g. `jane-doe.md`).

Keep whatever helps: role and where they fit, how we met, ongoing threads, preferences, follow-ups. Link to related notes in `02-projects/`, `03-areas/`, and `04-ideas/`.

Test: if a note is fundamentally *about a person*, it lives here. If it's about work that happens to involve them, file it with the work.
